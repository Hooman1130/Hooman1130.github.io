KATSU Questions Guide

Here are the answers to a bunch of commonly asked questions about KATSU. These questions are only questions that are not found in the FAQ channel of the KATSU discord server.

Questions & Answers

What's a module?:
A module is a JSON script that tells KATSU what content to scrape and display in the app. Anyone can make a module. Tutorials on how to make one can be found here. See this video to learn how to use modules correctly.

What's a Video Resolver/Server?:
A server/video resolver is what Katsu fetches the video file from. The option to select one pops up every time you click on an episode. Some common video resolvers include

  • Gogo-Play
  • Streamtape
  • Mixdrop
  • Dood
  • FCDN
  • VCDN
  • Decoded
Some resolvers are good, some are bad, some don't work, some don't work for downloads, some work great for downloads, some buffer, some don't. There's lots of variation between them.

What are tickets?:
Tickets allow you to download episodes/videos. 1 ticket = 1 download. You get tickets by clicking the "Get Tickets!" button in the downloads page and watching an ad. You get 3 tickets per ad.

How do I update a module?:
You update modules in the same page that you install them, the modules library. More instructions are below.

  1. Go to the modules manager in the settings or by pressing the globe icon in the upper right corner of the main page.
  2. Press the download icon in the upper right corner of the module manager to enter the module library.
  3. Tap the module you want to be updated to update it.

What is Ketsu?:
Ketsu is a new app that Orion is working on. It is basically KATSU and ZETSU combined, but it will also have many new features. Check out #info for frequent updates. It will probably come out in early 2021 for Apple Devices. Videos of Ketsu can be found here. Module Makers get early access to Ketsu.
